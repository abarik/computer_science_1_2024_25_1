Files, directories, compressing

Problem 1

In this section, you will learn how to create directories and simple text files with special symbols, and how to compress an entire directory structure.

  1. Select a proper base directory on your computer. For instance, your home directory or its Documements subdirectory would be fine. We call this base directory as BASEDIR. My choice is c:\Users\abarik\Documents\ on own computer or c:\student on classroom computer.

  2. Create the following directory structure under the BASEDIR.

basics_of_mathematics
computer_science_1
    lab_files
        documents
        data
        output
            image
            data
  1. Create a plain text file in computer_science_1/lab_files/documentsas special_characters.txt. Please type the following content into special_characters.txt. Save and close the file.
Fundamental arithmetic operations: + - / * ^

Common symbols are ' " ! % | / \  = () [] {} $ < > # & @ ? ; , . : 

You can skip this part:

- (hyphen) 
– (en dash or minus, Alt+0150) 
— (em dash, Alt+0151).

In R ~ (tilde) is used for formulas and ` (backtic) for non-syntactic variable names.
Tip

You can read more information about hyphen, minus, and dash:

  • Microsoft Writing Style Guide
  • Hyphen, minus, en-dash, and em-dash: difference and usage in English
  • Em Dash, En Dash, Hyphen: How to Use Correctly
  • Your keyboard doesn’t have an em dash symbol—here’s what to do

  1. Create a compressed zip file (computer_science_1.zip) from the contents of the computer_science_1 directory.

  2. Send the file computer_science_1.zip to the abari.kurzus@gmail.com. The subject of this email is Problem 1.

Problem 2

In this section we will learn how to compress and uncompress directories and how to copy and edit plain text files.

  1. Download the file problem_2.zip from here.

  2. Uncompress the file problem_2.zip into your BASEDIR.

  3. Create a new sub-folder output in directory problem_2.

  4. Copy quiz.txt from directory input to directory output as quiz_answer.txt.

  6. Save and close the file quiz_answer.txt.

  7. Create a compressed zip file (quiz.zip) from the contents of the problem_2 directory.

  8. Send the file quiz.zip to the abari.kurzus@gmail.com. The subject of this email is Problem 2.

Problem 3

In this section we will learn how to how to create file of any type.

  1. Create a new file_types directory in your BASEDIR directory.

  2. Create at least 5 files with simple, arbitrary content from the following file types. Creating 10 files means a Great answer badge (3 extra points). Save them in directory file_types:

  • TXT, simple text file (file.txt)
  • DOCX, formatted text file (file.docx)
  • ODT, formatted text file (file.odt)
  • PDF file (file.pdf)
  • PPTX, presentation file (file.pptx)
  • XLSX, spreadsheet file (file.xlsx)
  • ODS, spreadsheet file (file.ods)
  • HTML, web file (file.html)
  • PNG image file (file.png)
  • JPG image file (file.jpg)
  • ZIP, compressed file (file.zip)
  • BIB, bibliography file (file.bib)
  • OMV, jamovi file (file.omv)
Tip

The files above can usually be created under Windows with the following applications:

  • TXT - Notepad
  • DOCX - MS Word, LibreOffice Writer, Google Docs
  • ODT - LibreOffice Writer, MS Word, Google Docs
  • PDF - MS Word, LibreOffice Writer, Google Docs
  • PPTX - MS PowerPoint, LibreOffice Impress, Google Slides
  • XLSX - MS Excel, LibreOffice Calc, Google Sheets
  • ODS - LibreOffice Calc, MS Excel, Google Sheets
  • HTML - MS Word, LibreOffice Writer, Google Docs
  • PNG - Paint
  • JPG - Paint
  • ZIP - operating system built-in tool, WinZip
  • BIB - JabRef
  • OMV - Jamovi

You can read more information about file types and extensions:

  • Showing File Extensions
  • What are the most common file types and file extensions?
  • Common file name extensions in Windows
